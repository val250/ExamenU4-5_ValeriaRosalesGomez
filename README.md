# ExamenUnidad4 y 5_Patrones_ValeriaRosalesGomez  
## Valeria Rosales Gómez C21212361  

**Problema a resolver:**  
Simulación de procesos industriales automatizados

</div>
El código presentado muestra una simulación de un proceso industrial 
en donde se presentan un conjunto de materiales a reciclar y posteriormente el programa
determina si dicho material es o no es apto para reciclar. Para lograr esto
se aplicaron 5 patrones de diseño para su funcionamiento: Object Pool, Singleton,
Adapter, State y Capas.
